void LeddarNET::Device::StartDataTransfer ( DataLevels  aLevels)

Start transferring data from the device or record. If no connection or replay is active or the transfer was already started, an exception will be thrown.

Parameters
aLevelsA combination (OR) of all the levels that are required. Note that the State level is optional as it will always be transfered even if not requested in this parameter.
