int LeddarStartUpdate ( LeddarHandle  aHandle,
LeddarU32  aType,
char *  aFileName

Start a code update.

Must have a live connection, no data transfer, no recording and no other updater started.

Parameters
aHandleHandle returned by LeddarCreate.
aTypeType of code to update (LT_COMM_SOFTWARE_TYPE_*)
aFileNamePath and name of file to read to get the update.
Returns
Standard LeddarC return code.
